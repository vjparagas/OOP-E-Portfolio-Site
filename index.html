                    if (withdraw > balance) System.out.println("Not enough balance.");

import java.util.Scanner;
public class StudentAgeAnalyzerEvangelista {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Enter student name: "); String studentName = sc.nextLine();
        System.out.print("Enter age: "); int age = sc.nextInt();
        String category = getCategory(age);
        System.out.println("Student name: " + studentName);
        System.out.println("Age: " + age);
        System.out.println("Category: " + category);
        sc.close();
    }
    public static String getCategory(int age) {
        if (age >= 0 && age <= 12) return "Child";
        else if (age >= 13 && age <= 19) return "Teenager";
        else if (age >= 20 && age <= 59) return "Adult";
        else if (age >= 60) return "Senior Citizen";
        else return "Invalid Age";
    }
}

A simple wallet system that simulates basic financial operations such as depositing and withdrawing funds using OOP principles.

<> Source Code
import java.util.Scanner;
public class SimpleWalletEvangelista {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);
        int balance = 3000, choice, transactionCount = 0;
        while (true) {
            System.out.println("===== WALLET MENU =====");
            System.out.println("1. View Balance  2. Add Money  3. Spend Money  4. Exit");
            System.out.print("Enter choice: "); choice = input.nextInt();
            switch (choice) {
                case 1: transactionCount++; System.out.println("Balance = " + balance); break;
                case 2: System.out.print("Enter amount to add: "); int add = input.nextInt();
                    if (add > 0) { balance += add; transactionCount++; System.out.println("New Balance = " + balance); }
                    else System.out.println("Invalid amount."); break;
                case 3: System.out.print("Enter amount to spend: "); int spend = input.nextInt();
                    if (spend > balance) System.out.println("Not enough balance.");
                    else if (spend > 1000) System.out.println("Maximum spend is 1000 only.");
                    else if (spend <= 0) System.out.println("Invalid spend amount.");
                    else { balance -= spend; transactionCount++; System.out.println("New Balance = " + balance); } break;
                case 4: System.out.println("Successful Transaction Count = " + transactionCount); System.out.println("Thank you!"); break;
                default: System.out.println("Invalid choice.");
            }
            if (choice == 4) break;
        }
    }
}
Attachments
Reflection & Learnings

This activity taught me how to apply loops and conditions in a real program like a wallet system. I learned how to set rules for spending and deposits, making the program more practical and realistic. It was satisfying to see how the system responded correctly to user actions. This activity showed me that programming is not just about syntax, it's about applying logic to create programs that reflect real-life situations.

A personal expense tracker that records and summarizes spending using arrays and OOP concepts in Java.

import java.util.Scanner;
public class ExpenseTrackerEvangelista {
    public static void showTitle() { System.out.println("\n|--- Simple Expense Tracker ---|\n"); }
    public static double getTotal(double f, double t, double m) { return f + t + m; }
    public static String budgetCheck(double total, double budget) {
        return total > budget ? "Oof, you went over your budget!" : "Total is within your budget. Keep it up ^^";
    }
    public static void showResults(double total, double budget, String msg) {
        System.out.println("Total Expenses: " + total);
        System.out.println("Budget: " + budget);
        System.out.println("Budget Status: " + msg);
    }
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in); showTitle();
        System.out.print("Enter food expenses: "); double f = input.nextDouble();
        System.out.print("Enter travel expenses: "); double t = input.nextDouble();
        System.out.print("Enter other expenses: "); double m = input.nextDouble();
        System.out.print("Enter your budget: "); double budget = input.nextDouble();
        double total = getTotal(f, t, m);
        showResults(total, budget, budgetCheck(total, budget));
        double rem = budget - total;
        System.out.println(rem >= 0 ? "You still have PHP" + rem + " left." : "You overspent by " + Math.abs(rem) + ".");
        System.out.println("\nThanks for using VJ's Expense Tracker. Keep on grinding^^");
        input.close();
    }
}
Attachments
Reflection & Learnings

In Activity 5, I learned how to use methods in Java to make a program more organized and practical. I saw how combining user input with calculations can help track expenses and check if the budget is enough. It felt realistic because it connects programming with everyday life. This activity made me realize that coding can be used to manage money wisely and create tools that help people make better decisions.
